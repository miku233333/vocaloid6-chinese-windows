# Vocaloid 6 中文語音庫 - Windows 版

🎤 **Vocaloid 6 中文語音庫安裝腳本 (Windows)**

## 📦 內容

| 文件/文件夾 | 說明 |
|------------|------|
| `install.bat` | 一鍵安裝腳本 |
| `uninstall.bat` | 卸載腳本 |
| `data/` | 語音庫數據 |
| `installer/` | 安裝器 |
| `scripts/` | 輔助腳本 |

## 🚀 安裝方法

### 方法 1: 一鍵安裝
```batch
install.bat
```

### 方法 2: 手動安裝
1. 以管理員身份運行 CMD
2. 執行 `install.bat`
3. 跟隨提示完成安裝

## ❌ 卸載

```batch
uninstall.bat
```

## 📋 系統要求

- **操作系統**: Windows 10/11 (64-bit)
- **Vocaloid 6**: 已安裝
- **磁碟空間**: 至少 2GB
